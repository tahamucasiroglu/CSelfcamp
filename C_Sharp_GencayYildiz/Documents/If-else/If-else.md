# If-Else

### Tek satýr if else

```
if(durum)
{
	kod 1;
}
else
{
	kod 2;
}
```

bu kodu 

```
durum ? kod1 : kod2;
```

þeklinde yazýlabilir. Geniþletmek istersek

```
if(durum1)
{
	kod 1;
}
else if(durum2)
{
	kod 2;
}
else
{
	kod 3;
}
```

bunu

```
durum ? kod 1 : durum ? kod 2 : kod 3;
```

olarak yazabiliriz.
